# Changelog

All notable changes to the Developers Toolkit will be documented in this file.

## [3.5.2] - 2026-04-16
### Changed
- **Rebranding**: Complete transition from `claude-plugin` to `developers-toolkit`.
- **Focus**: Optimized toolkit exclusively for **WordPress and React** teams.
- **Workflow**: Renamed GitHub Action templates to `developers-*` prefix.

### Removed
- Removed wordpress-specific context skills and documentation.
- Deleted `developers/skills/wordpress-context` directory.

### Fixed
- Fixed broken internal links in README and Installation guides.
- Synchronized version numbers across all manifests and badges.
- Corrected repository paths in marketplace commands.

## [3.5.0] - Initial Consolidated Release
- Merged multiple agent and command repositories into a single unified plugin.
- Added 23 specialized agents and 34 commands.
- Integrated automated hooks for PHP and JS linting.
