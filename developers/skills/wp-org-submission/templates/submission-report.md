## Submission Instructions

1. Create production zip (excluding dev files):
   ```bash
   cd /path/to/plugins
   zip -r {plugin-slug}.zip {plugin-slug}/ \
     --exclude "*/.git/*" \
     --exclude "*/node_modules/*" \
     --exclude "*/tests/*" \
     --exclude "*/phpunit.xml" \
     --exclude "*/package.json" \
     --exclude "*/package-lock.json" \
     --exclude "*/.eslintrc*" \
     --exclude "*/.phpcs.xml" \
     --exclude "*/Gruntfile.js" \
     --exclude "*/webpack.config.js"
   ```

2. Verify zip size:
   ```bash
   du -sh {plugin-slug}.zip  # Must be under 10MB
   ```

3. Submit at: https://wordpress.org/plugins/developers/add/

4. Expected review time: 1–14 business days

5. Check for email from `plugins@wordpress.org` — whitelist this address now
